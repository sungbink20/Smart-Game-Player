# Smart-Game-Player

A browser-based General Game Playing (GGP) agent built with JavaScript and Stanford’s Gamemaster and Epilog frameworks.
This player, bombini_gusini, implements adaptive decision-making logic using both Monte Carlo Tree Search (MCTS) and Iterative Deepening Minimax algorithms. It dynamically analyzes branching factors and adjusts exploration parameters to select optimal strategies during gameplay. The interface connects to the Stanford Gamemaster platform and logs gameplay results through a local transcript display.
